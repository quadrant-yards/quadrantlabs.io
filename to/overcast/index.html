▶ Manual setup
  1. 1Open Shortcuts and tap + to create a new shortcut
  2. 2Add Split Text — input: Shortcut Input, separator: /id
  3. 3Add Get Item from List — Last Item
  4. 4Add Split Text — input: Item from List, separator: ?
  5. 5Add Get Item from List — First Item
  6. 6Add URL — value: https://overcast.fm/+itunes[Item from List]
  7. 7Add Open URLs
  8. 8Tap the shortcut name → enable Show in Share Sheet → set input type to URLs
